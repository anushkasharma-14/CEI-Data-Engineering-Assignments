# Week 3 Assignment – SQL Analysis Using Subqueries, CTEs, and Window Functions

## Overview

This week's assignment focuses on analyzing the Superstore dataset using advanced SQL concepts such as Subqueries, Common Table Expressions (CTEs), and Window Functions. The dataset was imported into MySQL and organized into separate tables for customers, products, and orders to perform business-oriented sales analysis.

## Topics Covered

* Subqueries
* Common Table Expressions (CTEs)
* Window Functions (`RANK()`, `ROW_NUMBER()`)
* Customer Sales Analysis
* Sales Aggregation and Ranking
* JOIN Operations
